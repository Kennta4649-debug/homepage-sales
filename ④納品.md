# STEP4｜ホームページを仕上げて納品する

## 反響があったらやること

### 1. ヒアリング
ヒアリングシート.md の項目を確認する。
DMかLINE・電話で聞く。

確認すること：
- [ ] デモからどこを変えたいか
- [ ] 追加したいページ・機能はあるか
- [ ] 写真は用意できるか（できなければ撮影手配）
- [ ] ロゴはあるか
- [ ] 公開希望日はいつか
- [ ] 料金・支払い方法の合意

### 2. 本制作
- デモをベースにWixで仕上げる
- 実際の写真・テキストに差し替える
- スマホ表示を確認
- ページタイトル・説明文（SEO）を設定

### 3. 確認してもらう
```
お世話になっております。
サイトが完成しましたのでご確認をお願いします。

確認URL：【URL】

修正があればお気軽にお申し付けください。
（修正回数：【〇回まで無料】など決めておく）
```

### 4. 公開・納品
- Wixのドメイン設定（独自ドメインが必要な場合は案内）
- 公開ボタンを押す
- 納品連絡を送る

```
【店名】様

ホームページを公開しました！🎉

URL：【URL】

ご不明な点があればいつでもご連絡ください。
ご依頼いただきありがとうございました。
```

---

## 料金の目安（参考）

| プラン | 内容 | 価格例 |
|---|---|---|
| ライト | デモをそのまま仕上げ（5ページ以内） | 3〜5万円 |
| スタンダード | ヒアリングから制作（8ページ以内） | 5〜10万円 |
| 写真撮影込み | 上記＋出張撮影 | +1〜2万円 |

※価格は自由に設定してください。
