# STEP1｜ホームページのない店を探す

## ターゲット条件
- 個人店（チェーン・フランチャイズは除く）
- インスタグラムをやっている
- ホームページがない（または食べログ・ホットペッパーだけ）

---

## インスタで探す方法

### ハッシュタグ検索
```
#エリア名+業種 で検索する
例：#梅田カフェ　#難波ランチ　#天王寺美容室　#心斎橋居酒屋
```

### プロフィール欄で確認
| プロフィールのリンク | 判定 |
|---|---|
| リンクなし | ターゲット ✅ |
| リットリンク・linktreeのみ | ターゲット ✅ |
| 食べログ・ホットペッパーのみ | ターゲット ✅ |
| 独自ドメインのサイトあり | 除外 ❌ |

### 個人店かどうかの見分け方
- 投稿がオーナー自身の言葉で書かれている
- 「手作り」「こだわり」「オーナーの〇〇です」などの文言
- フォロワー数が数百〜1万程度
- チェーンっぽい統一されたビジュアルは除外
