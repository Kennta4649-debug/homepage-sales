# STEP3｜挨拶文とデモサイトを送る

## 送る方法
- インスタのDM（一番返信率が高い）
- プロフィールにメールがあればメールも可

---

## インスタDM テンプレート

### パターンA｜シンプル版
```
はじめまして、【自分の名前】と申します。

【店名】さんのインスタをいつも拝見しています。
【具体的にほめる一言：例「〇〇の写真がとても美味しそうで」「センスのある雰囲気が素敵で」】

ご縁があればと思い、試しにホームページのデモを作ってみました。
よろしければご覧ください👇

【デモサイトのURL】

気に入っていただけたら、ご要望に合わせて仕上げます。
費用や内容についてはお気軽にご相談ください。

突然のご連絡、失礼いたしました🙏
```

### パターンB｜効果を伝える版
```
はじめまして、【自分の名前】と申します。

【店名】さんのインスタ、【具体的にほめる一言】、素敵だなと思って拝見していました。

インスタは既にお使いなのに、Googleで検索しても出てこないのはもったいないと思い、
試しにホームページのデモを作ってみました。

👇こんな感じのサイトです
【デモサイトのURL】

ホームページがあると：
・Googleで「【エリア】【業種】」と検索したお客さんに見つけてもらえる
・営業時間・メニューの問い合わせが減る
・インスタを見た人が「信頼できる店」と感じやすくなる

ご興味あれば、費用など含めてお気軽にご相談ください😊
突然のご連絡、失礼いたしました。
```

---

## 送るときのコツ

- **インスタの投稿を具体的にほめる**（「素敵ですね」より「〇月の〇〇の投稿が」）
- **最初から料金を出さない**（興味を持ってもらってから）
- **デモURLは短く見やすく**（Wixのデモリンクをそのまま）
- **絵文字は1〜2個**（多すぎると業者っぽく見える）
- **送る時間帯**：平日の10〜12時 or 14〜17時が返信率が高い

---

## 返信があったときの対応

### 「興味あります」と来たら
→ STEP4へ。ヒアリングシートの内容を確認していく。

### 「料金はいくらですか？」と来たら
```
ありがとうございます！
内容によって変わりますが、【〇〇円〜】で対応しています。
一度、どんなサイトにしたいかお聞きできますか？
```

### 「今は必要ないです」と来たら
```
そうですか、承知しました！
またいつかご縁があればよろしくお願いします😊
```
（しつこくしない。次の候補へ）
