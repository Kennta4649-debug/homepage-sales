# STEP2｜店の情報を集めてデモサイトを作る

インスタから読み取れる情報だけでデモを作る。
店主に連絡する前に作るので、公開情報だけ使う。

---

## インスタから集める情報

- [ ] 店名
- [ ] 業種・ジャンル
- [ ] 住所（プロフィール or 投稿から）
- [ ] 電話番号（プロフィールに載っていれば）
- [ ] 営業時間・定休日（投稿やストーリーから）
- [ ] 看板メニュー・人気商品（投稿から）
- [ ] お店のこだわり・雰囲気を表す言葉（キャプションから）
- [ ] 写真（スクリーンショットでOK、デモ用に使う）

---

## Wix Harmony に送るプロンプト

以下の`【 】`を埋めてWix HarmonyのAIに貼り付ける。

```
【店名】のホームページのデモを作ってください。

業種：【業種】
場所：【住所・エリア】
営業時間：【営業時間】※不明なら省略
定休日：【定休日】※不明なら省略

お店の特徴（インスタの投稿より）：
【キャプションから読み取ったこだわり・雰囲気を2〜3文】

載せる内容：
- トップページ（キャッチコピーとお店の写真）
- メニュー・サービス紹介
- アクセス・営業時間
- お問い合わせ

デザイン：【インスタの雰囲気に合わせた色・テイスト】
日本語サイトにしてください。
シンプルで見やすいレイアウトにしてください。
```

---

## 業種別の追加指示

### 飲食店
```
食欲をそそる写真が映えるレイアウトにしてください。
予約・お問い合わせボタンを目立たせてください。
```

### 美容・サロン
```
清潔感とおしゃれさが伝わるデザインにしてください。
料金表とご予約のセクションを入れてください。
```

### 小売・雑貨
```
商品が映える余白のあるレイアウトにしてください。
「商品一覧」か「おすすめ商品」のセクションを入れてください。
```

### 士業・専門職
```
信頼感・誠実さが伝わるシンプルなデザインにしてください。
「よくある質問」と「お問い合わせフォーム」を入れてください。
```

---

## デモ完成後の確認チェック

- [ ] 店名・住所・電話番号に誤りがないか
- [ ] メニューや料金の情報は正確か（不明な箇所は「お問い合わせください」にする）
- [ ] 写真はデモ用であることを明記するか削除
- [ ] スマホ表示でも崩れていないか
- [ ] デモサイトのURLをコピーしておく
